# contractor-os
A Python CLI-based system for tracking contractor jobs, calculating profit, and managing business data.
# Features 
- Add new jobs with pricing and cost details
- Automatically calculate profit
- Search jobs by name
- Clear all stored jobs
- View total profit
- JSON Memory
# Tech Used
- Python
- Command Line Interface (ClI)
